Student Management System

A Win32 application for managing classes of students and their records.

I created the Student Management System (SMS) for my Computing A-Level final project.

The program was written in Pascal, and implemented with Delphi and Microsoft ADO DB for the database backend. It was created according to the needs and wishes of the target user, a local French tutor, and I followed the complete system maintenance cycle when realising the project.

The Write-up

As part of the project, I had to document every part of the work - from initial analysis of current systems and the potential users, to designing the data and interface, creating test plans and also detailing the user's feedback.

Including the code listing for the project, my writeup comes to just over two hundred pages long, as well as an additional user manual which guides the user through the program's various features and settings. If you wish to obtain a copy of these, please email me by contacting me.
