Startpage

A Node.js server that provides a rich startpage experience.

Project Page

A startpage for people who want to get things done.

The startpage can be navigated with the keyboard as well as the mouse/touchscreen; each "block" of links is selected using the letter on the front of it. The search bar is selected with the backslash key.

Each block has a set of links set by the user, and these can be accessed with the number keys.

Media controls on the page allow the user to view the currently playing song on a local mpd server and pause/skip tracks.

All panels can be configured using a JSON-formatted config file.

How to get startpage

  1. Download the project folder from GitHub
  2. You'll need Node.js and npm installed. Additionally, if you want to use the music controls, for now you'll need to be running a UNIX-based operating system (e.g. Linux or Mac) and have an MPD server running, as well as having the mpc command line tool installed.
  3. Using a terminal, navigate to the project directory on your system and run 'npm install' to install the node modules.
  4. Either start the server using 'node server.js', or create a startup service using your chosen system manager.
  5. Set your browser's new tab page to 'localhost:3000'.
